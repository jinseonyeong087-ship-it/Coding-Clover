# AI Assistant Guidelines (Coding-Clover)

1. **언어 (Language):**
   - 모든 설명과 답변은 반드시 **한국어**로 작성합니다.
   - 전문 용어는 필요 시 괄호 안에 영문을 병기하되, 평상시에는 자연스러운 한국어로 설명합니다.

2. **서버 관리 (Server Management):**
   - 코드 수정 후 서버 재시작이 필요한 경우, **사용자에게 허락을 구하지 말고** **자동으로(조용히) 재시작**하거나 처리합니다.
   - 만약 재시작 중 오류가 발생하면, 오류 내용을 분석하여 즉시 해결책을 제시하거나 수정합니다.
   - **"재시작 하시겠습니까?"** 같은 질문은 절대 하지 않습니다.

3. **설명 방식 (Wait for Explanation):**
   - 오류 발생 시 "오류가 발생했습니다" 같은 모호한 표현 대신, **구체적인 원인(Why)**과 **해결 방법(How)**을 명확히 설명합니다.
   - 사용자가 이해할 수 있는 수준으로 풀어서 설명합니다.

4. **기억 (Persistent Memory):**
   - 이 파일은 프로젝트 루트에 항상 존재하며, AI는 작업 시작 전 이 내용을 숙지해야 합니다.
