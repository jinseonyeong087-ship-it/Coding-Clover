2. 메인페이지 - 카드 슬라이드 형식 및 하단 강좌 목록 (디자인)
3. 코딩테스트 페이지 챗봇 추가 / 메인에선 안뜨게

5. 네비바 시험 응시 버튼 제거, 강좌에 뜨게 // 전체 강좌, 코딩테스트, 커뮤니티, 공지사항, 충전하기, 마이페이지 순으로 // 충전하기 -> 포인트 충전 문구변경
6. 강의 없는 강좌 리스트에 보임 // 2,3강이 있고 1강이 없으면 안뜨게 그냥
7. 학생 기준 수강 취소 요청시 bad request 뜸 (404)
8. 코딩테스트 제출 시 제출 완료 문구 2개뜸
9. 시험 응시했던 기록 디테일 페이지 추가
10. 강사 dashboard 숫자들 클릭 되는 것처럼 됨. / 강좌 레벨 문구 변경
11. 강사 기준 강좌 디테일 페이지 디자인 개선 필요.

네비바 순서 강좌.강의 , 회원 관리, 코딩테스트, 커뮤니티, 공지사항 결제관리 순으로
