# 코딩 학습 플랫폼 기능 정의서  
*(ETU 프로젝트 기준)*

---

## 1. 시스템 공통 기능 (플랫폼 레벨)

### 1.1 강좌 / 강의 구조
- 하나의 **강좌(Course)** 안에 여러 개의 **강의(Lecture)** 구성
- 레벨별 강좌 구분
  - 입문
  - 초급
  - 중급
  - 고급
- 강의는 순차 수강 구조 지원
- 강좌 단위로 수강 권한 관리

---

### 1.2 결제 시스템
- 결제 방식 다중화
  - 크레딧 결제
  - 정기 수강권(구독형)
  - 1회 수강권
- 크레딧 / 구독 상품 차별화 구성
- 신규 가입 시 소정의 크레딧 또는 1회 수강권 지급
- 결제 이력 관리
- 환불 이력 관리

---

### 1.3 Q/A 시스템
- 수강생 질문 등록
- 강사 답변 등록
- 질문 상태 관리
  - 대기
  - 답변 완료
- 답변 수정 이력 관리
- 알림 연동 (답변 등록 시)

---

### 1.4 커뮤니티 기능
- 수강생·강사 참여 커뮤니티
- 게시글 / 댓글 기능
- 신고 기능 (관리자 처리)
- 관리자 제재 기능
  - 숨김
  - 삭제

---

### 1.5 회원 전환 및 승인 정책
- 강사 회원가입 시 관리자 승인 필수
- 수강생 → 강사 전환 로직 지원
- 승인 / 반려 이력 관리
- 승인 결과 알림 제공

---

### 1.6 스케줄 관리
- 수강생 / 강사 공통 스케줄 관리 탭
- 강의 일정 표시
- 시험 일정 표시
- 개인 학습 일정 관리

---

### 1.7 강의 업로드 및 승인 흐름
- 강사는 강의 업로드 **요청만 가능**
- 관리자 승인 후 강의 공개
- 승인 완료 후 강사는 강의 수정 불가
- 관리자 문의 프로세스 제공

---

### 1.8 소셜 로그인
- 카카오 로그인
- 구글 로그인
- 일반 회원가입 병행 지원

---

### 1.9 알림 시스템 *(보완 필수)*
- 강의 승인 / 반려 알림
- Q/A 답변 알림
- 시험 응시 가능 알림
- 공지사항 알림

---

### 1.10 로그 및 이력 관리 *(보완 필수)*
- 수강 로그
- 출석 로그
- 결제 / 환불 로그
- 시험 응시 이력
- 관리자 승인 / 반려 이력

---

### 1.11 권한 정책 *(보완 필수)*
- 역할별 접근 권한 정의
  - 수강생
  - 강사
  - 관리자
- 승인 전·후 접근 권한 차등
- 수강 종료 후 콘텐츠 접근 정책 명시

---

### 1.12 검색 / 필터 / 정렬 / 페이징 기능 *(필수)*

#### 검색 대상
- 강좌 / 강의 검색
- 강사 검색
- Q/A 검색
- 커뮤니티 게시글 검색
- 관리자용 회원 검색

#### 검색 방식
- 키워드 기반 검색
- 부분 일치 검색 지원

#### 필터 기능
- 레벨 필터
  - 입문 / 초급 / 중급 / 고급
- 카테고리 필터
- 진행 상태 필터
  - 수강 가능
  - 수강 중
  - 종료
  - 승인 대기

#### 정렬 기능
- 최신순
- 인기순
- 평점순
- 조회수순

#### 페이징 처리
- 목록 페이징 처리
- 페이지 단위 데이터 조회

## 2. 수강생 기능

### 2.1 회원가입 및 레벨 측정
- 회원가입 시 코딩 테스트로 초기 레벨 측정
- 테스트 결과 기반 추천 강좌 제공

---

### 2.2 학습 기능
- 실 코딩 문제 풀이 기능
- 레벨별 강의 수강
- 강의 시청 / 문제 풀이 시 출석 인정

---

### 2.3 레벨업 시스템
- 강의 80% 이상 이수 시 레벨업 시험 응시 가능
- 시험 통과 시 레벨 상승
- 재응시 정책 관리

---

### 2.4 출석 관리
- 강의 시청 시 출석 처리
- 코딩 문제 풀이 시 출석 처리
- 출석 기록 조회 가능

---

### 2.5 강의 평가
- 강의 만족도 평가 기능
- 평점 및 후기 등록

---

### 2.6 학습 대시보드 *(보완 필수)*
- 현재 레벨 표시
- 강의 이수율
- 시험 응시 가능 상태
- 추천 강의 안내

---

### 2.7 시험 결과 피드백 *(보완 필수)*
- 통과 / 미통과 여부 제공
- 점수 제공
- 오답 요약
- 재응시 가능 시점 안내

---

## 3. 강사 기능

### 3.1 강사 회원 관리
- 강사 회원가입
- 관리자 승인 후 활동 가능
- 계좌 정보 등록
- 커미션 지급 기준 관리

---

### 3.3 과제 관리
- 강사 주관 코딩 과제 작성
- 과제 제출 관리
- 과제 평가

---

### 3.4 Q/A 관리 *(보완)*
- 질문 목록 관리
- 답변 등록 / 수정
- 답변 상태 관리
- FAQ 전환 가능

---

### 3.5 강사 대시보드 *(보완 필수)*
- 개설 강좌 현황

- 수익 요약
- 승인 상태 확인
