# 프로젝트 기능 명세서

## 공통 / 회원가입 및 인증 (Common / Auth)
- 회원가입 (학생/강사)
- 로그인
- 아이디 찾기
- 비밀번호 찾기
- 이메일 인증
- 프로필 관리
- 통합 검색

## 학생 (Student)
- 강좌 목록 조회
- 강좌 상세 정보 확인
- 강좌 수강 신청 (무료/유료)
- 포인트 충전 및 결제
- 포인트 사용 내역 조회
- 강의 시청 (동영상)
- 학습 진도율 확인
- 강의별 질의응답(Q&A) 질문 등록 및 조회
- 커뮤니티 게시글 작성/수정/삭제/조회
- 댓글 작성
- 코딩 테스트 문제 목록 조회
- 코딩 테스트 문제 풀이 및 코드 실행
- 코드 제출 및 자동 채점
- 제출 이력 확인
- 시험 응시
- 시험 결과 및 점수 확인
- AI 챗봇 튜터링 (질문하기)
- 마이페이지 (내 강의, 포인트, 활동 내역 관리)

## 강사 (Instructor)
- 강사 대시보드 (현황 확인)
- 강좌 생성 및 개설 신청
- 강좌 정보 수정 및 커리큘럼 관리
- 강의 영상 및 자료 업로드
- 시험 문제 출제 및 생성
- 시험 관리
- 수강생 질의응답(Q&A) 답변 관리
- 강사 프로필 관리

## 관리자 (Admin)
- 관리자 대시보드 (통계)
- 회원 관리 (학생/강사 상세 조회)
- 강사 승인/반려 관리
- 강좌 승인/반려 관리
- 전체 강의 콘텐츠 관리
- 결제 내역 관리
- 수강 신청 내역 관리
- 시험 관리
- 공지사항 작성/수정/삭제 관리
